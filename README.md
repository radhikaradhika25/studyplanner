#STUDY PLANNER

## Description

The Study Planner  is a web-based application that helps students organize their study tasks. Users can add subjects, topics, and deadlines, and manage their study schedule efficiently.

## Features

* Add study tasks
* Manage subjects and topics
* Set deadlines
* Store data using ShaktiDB
* Flask backend integration

## Technologies Used

* HTML
* CSS
* JavaScript
* Python Flask
* ShaktiDB

## Project Structure

* index.html
* style.css
* script.js
* app.py
* db_test.py

## How to Run

1. Activate virtual environment
2. Install dependencies using:
   pip install -r requirements.txt
3. Run the application:
   python app.py
